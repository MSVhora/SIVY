UI Design and Iconography for a Global Telecom major

The Challenge


Design a clean and mobile-friendly UI that integrates a plethora of functionalities and features and a complex sitemap that breaks down to 7-levels of information workflow.


The web-based configuration software required a major overhaul not just in the GUI, but also in the information architecture and data workflow, that made it easy for the user to view multiple information blocks in a single view.

The Solution

With skeletal wireframes built on in-depth use-case scenarios and information workflow studies, Niyati started the UI Design process with the key functionalities in mind. The sitemap was simplified by grouping information packets and representing them with intuitive icons and an expandable menu framework and tool tips built on JQuery and Angular JS.

Key Tools and Technologies


  • HTML5 & CSS3
  • Bootstrap with AngularJS
